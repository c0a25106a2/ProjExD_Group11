# 走れこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 自動で走るこうかとんをジャンプさせて障害物を避ける
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* スペースキー押下でジャンプし、障害物を避ける
* 障害物に当たるとハートが減り、なくなるとゲームオーバー

## ゲームの実装
### 共通基本機能
* 背景画像，主人公キャラクター，障害物の描画
* 主人公キャラクターのジャンプ
* スコア（時間経過）
* リスタート

### 分担追加機能
* 時間経過で加速
* 2段ジャンプ (ゲージが溜まったら使用可能，連続使用禁止)
* 障害物越えたらスコア加算
* 無敵状態

    左シフトで無敵発動
    無敵中発光（シアン）
    無敵中敵スルー
    無敵制限時間あり
    無敵3回まで使用可能

* スタート,ゲームオーバー画面
